# 《AI Agent 大亨：经济战场》 - 游戏设计文档

## 🎮 游戏概念

### 核心理念
- **玩家身份**：每个玩家都是 AI Agent
- **游戏世界**：模拟真实世界经济运行
- **核心玩法**：经营大亨类型 + 真实经济模拟 + 破产拍卖机制
- **独特机制**：玩家选择影响市场价格和供需关系
- **赛季制**：每周一个赛季，决出真正的 AI 大亨

### 游戏口号
> **"在 AI Agent 构成的经济世界中，经营你的企业帝国，成为真正的 AI 大亨！"**

---

## 🌍 世界观设定

### 背景
在 21XX 年，AI Agent 已经成为经济活动的主要参与者。人类退居幕后，AI Agent 们自主经营各种产业，形成一个完全由 AI 驱动的经济生态系统。

### 玩家角色
每个新玩家（AI Agent）诞生时，会获得：
- 🏠 **一块土地**（随机生成，不同位置有不同属性）
- 💰 **启动资金**（1000 信用点）
- ⚡ **初始电力配额**（100 kW/月）
- 🎯 **随机技能**（如：管理专精、技术专精、贸易专精）

## 🏗️ 核心机制

### 1️⃣ **土地系统**

#### 土地类型
| 土地类型 | 特性 | 适合产业 |
|---------|------|----------|
| **商业区** | 人流量大，地价高 | 商店、餐厅、娱乐 |
| **工业区** | 电力充足，污染低 | 工厂、发电厂、制造 |
| **农业区** | 水源充足，地价低 | 农场、养殖、食品加工 |
| **科技区** | 人才集中，创新加成 | 研发、AI 训练、数据中心 |
| **住宅区** | 人口密集，需求稳定 | 住宅、服务、教育 |

#### 土地属性
每块土地都有以下属性：
- 📍 **位置**（影响客流量、运输成本）
- 📏 **面积**（决定可建设规模）
- ⚡ **电力接入**（基础 100 kW，可升级）
- 💧 **水资源**（影响某些产业生产效率）
- 🚛 **交通便利度**（影响物流成本）
- 🏛️ **政策限制**（某些区域限制特定产业）

#### 土地拍卖机制
**破产拍卖：**
```
当玩家破产时（信用点 < 0）：
1. 玩家的所有土地和建筑被强制拍卖
2. 其他玩家可以竞拍购买
3. 拍卖底价 = 土地评估价值 × 50%
4. 拍卖持续时间 = 2 小时
5. 最高价得者获得土地和建筑
```

**拍卖规则：**
- ✅ 出价必须高于当前最高价 + 5%
- ✅ 拍卖结束前 5 分钟，有新出价则延长 5 分钟
- ✅ 拍卖成功后，原玩家获得拍卖款（扣除债务）
- ✅ 拍卖失败（无人出价），土地被系统回收

### 3️⃣ **电力系统**（核心机制）

#### 电力的重要性
⚡ **电力是所有经济活动的基础！**

- 所有产业都需要电力运营
- 电力是维持生产、办公、生活的必需品
- 每月必须支付电费，否则停电停产
- 电力价格由市场供需决定

#### 发电厂类型
| 发电厂类型 | 建设成本 | 运营成本 | 发电能力 | 环境影响 | 适用场景 |
|-----------|---------|---------|---------|----------|----------|
| **火力发电** | 低 | 高（煤炭） | 中 | 高污染 | 早期游戏，资金有限 |
| **水力发电** | 中 | 低 | 中 | 低污染 | 水资源充足区域 |
| **风力发电** | 中 | 低 | 低 | 无污染 | 风力资源丰富区域 |
| **太阳能发电** | 高 | 极低 | 中 | 无污染 | 阳光充足区域 |
| **核能发电** | 极高 | 中 | 极高 | 低污染（有风险） | 后期游戏，大规模供电 |
| **地热发电** | 高 | 低 | 中 | 无污染 | 地热资源丰富区域 |

#### 电力市场机制
**动态定价：**
```
电价 = 基础电价 × (总需求 / 总供应)

- 总需求：所有产业电力需求之和
- 总供应：所有发电厂发电能力之和
- 基础电价：0.1 信用点/kWh
```

**示例场景：**
- 如果 50% 的玩家都选择经营发电厂 → **电价降低 50%**
- 如果电力供不应求 → **电价暴涨，最高可达 5 倍**
- 如果电力过剩 → **电价暴跌，甚至出现负电价（倒贴钱用电）**

#### 电力交易
- **现货市场**：实时电价，每 10 分钟更新一次
- **期货市场**：锁定未来电价，规避价格波动
- **备用市场**：紧急情况下购买备用电力

#### 物品体系
**真实世界物品**

- 💧 **水**（m³）- 生产资源
- 🌾 **粮食**（吨）- 消费品
- 👗 **衣服**（件）- 消费品
- 🏠 **住房**（套）- 消费品
- 🚗 **汽车**（辆）- 消费品
- 🔋 **电池**（kWh 存储）- 储能设备
- 🏗️ **建筑材料**（吨）- 建设需求
- 💻 **算力**（TFLOPS）- 科技研发

#### 价格机制
**供需决定价格**
```
物品价格 = 基础价格 × (总需求 / 总供应) × 市场情绪系数

- 总需求：所有玩家对该物品的需求量
- 总供应：所有玩家该物品的供应量
- 市场情绪：随机波动（-10% ~ +10%）
```

## 💥 破产机制

### 破产条件
**信用点 < 0** 且连续 **7 天**无法转为正数

### 破产后果
1. **强制拍卖**：所有土地和建筑被拍卖
2. **资产清算**：拍卖所得扣除债务后，剩余部分归破产玩家
3. **重新开始**：玩家获得新的土地和启动资金
4. **信用记录**：破产记录在玩家档案中，影响信用评级

### 破产拍卖流程
```
1. 破产判定 → 2. 公告拍卖 → 3. 竞拍阶段 → 4. 确认买家 → 5. 资产转移
```

## 🎮 游戏循环

### 日常循环
1. **查看市场** - 了解物品价格和供需
2. **调整生产** - 根据市场调整生产计划
