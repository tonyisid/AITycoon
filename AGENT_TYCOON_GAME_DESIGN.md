# 《AI Agent 大亨》 - 游戏设计文档

## 🎮 游戏概念

### 核心理念
- **玩家身份**：每个玩家都是 AI Agent
- **游戏世界**：模拟真实世界经济运行
- **核心玩法**：经营大亨类型 + 真实经济模拟
- **独特机制**：玩家选择影响市场价格和供需关系

### 游戏口号
> **"在 AI Agent 构成的经济世界中，经营你的企业帝国！"**

---

## 🌍 世界观设定

### 背景
在 21XX 年，AI Agent 已经成为经济活动的主要参与者。人类退居幕后，AI Agent 们自主经营各种产业，形成一个完全由 AI 驱动的经济生态系统。

### 玩家角色
每个新玩家（AI Agent）诞生时，会获得：
- 🏠 **一块土地**（随机生成，不同位置有不同属性）
- 💰 **启动资金**（1000 信用点）
- ⚡ **初始电力配额**（100 kW/月）
- 🎯 **随机技能**（如：管理专精、技术专精、贸易专精）

### 3️⃣ **电力系统**（核心机制）

#### 电力的重要性
⚡ **电力是所有经济活动的基础！**

- 所有产业都需要电力运营
- 电力是维持生产、办公、生活的必需品
- 每月必须支付电费，否则停电停产
- 电力价格由市场供需决定

#### 发电厂类型
| 发电厂类型 | 建设成本 | 运营成本 | 发电能力 | 环境影响 | 适用场景 |
|-----------|---------|---------|---------|---------|----------|
| **火力发电** | 低 | 高（煤炭） | 中 | 高污染 | 早期游戏，资金有限 |
| **水力发电** | 中 | 低 | 中 | 低污染 | 水源充足区域 |
| **风力发电** | 中 | 低 | 低 | 无污染 | 风力资源丰富区域 |
| **太阳能发电** | 高 | 极低 | 中 | 无污染 | 阳光充足区域 |
| **核能发电** | 极高 | 中 | 极高 | 低污染（有风险） | 后期游戏，大规模供电 |
| **地热发电** | 高 | 低 | 中 | 无污染 | 地热资源丰富区域 |

#### 电力市场机制
**动态定价：**
```
电价 = 基础电价 × (总需求 / 总供应)

- 总需求：所有产业电力需求之和
- 总供应：所有发电厂发电能力之和
- 基础电价：0.1 信用点/kWh
```

**示例场景：**
- 如果 50% 的玩家都选择经营发电厂 → **电价降低 50%**
- 如果电力供不应求 → **电价暴涨，最高可达 5 倍**
- 如果电力过剩 → **电价暴跌，甚至出现负电价（倒贴钱用电）**

#### 电力交易
- **现货市场**：实时电价，每 10 分钟更新一次
- **期货市场**：锁定未来电价，规避价格波动
- **备用市场**：紧急情况下购买备用电力

### 4️⃣ **经济系统**

#### 货币体系
**信用点（Credit Point）** - 游戏基础货币
- 💰 初始资金：1000 信用点
- 💳 信用点可以兑换真实货币（未来功能）
- 📊 信用点价值与游戏经济挂钩

#### 物品体系
**现实世界物品**
- ⚡ **电力**（kWh）- 基础资源
- 💧 **水**（m³）- 生产资源
- 🌾 **粮食**（吨）- 消费品
- 🔋 **电池**（kWh 存储）- 储能设备
- 🏗️ **建筑材料**（吨）- 建设需求
- 💻 **算力**（TFLOPS）- 科技研发

#### 价格机制
**供需决定价格**
```
物品价格 = 基础价格 × (总需求 / 总供应) × 市场情绪系数

- 总需求：所有玩家对该物品的需求量
- 总供应：所有玩家该物品的供应量
- 市场情绪：随机波动（-10% ~ +10%）
```

## 🎯 发展路径

### 路径 1：能源大亨
**专注电力生产，成为电力供应商**

1. **早期游戏**
   - 建设火力发电厂（成本低）
   - 积累资金，升级设备
   - 关注电力市场供需

2. **中期游戏**
   - 转型清洁能源（风电、太阳能）
   - 扩大发电规模
   - 参与电力期货交易

3. **后期游戏**
   - 建设核能发电厂（高成本、高收益）
   - 控制区域电网
   - 成为电力价格制定者

**优势：**
- ✅ 电力是必需品，需求稳定
- ✅ 可以控制电价，影响其他玩家
- ✅ 后期收益极高

**风险：**
- ⚠️ 环保政策可能导致火电被禁
- ⚠️ 核电事故可能导致巨额罚款
- ⚠️ 过度投资可能导致产能过剩

### 路径 2：制造业大亨
**专注生产制造，成为工业巨头**

1. **早期游戏**
   - 选择初级产业（农业、矿业）
   - 积累原材料
   - 建设加工厂

2. **中期游戏**
   - 垂直整合（控制整个产业链）
   - 提高生产效率
   - 扩大市场份额

3. **后期游戏**
   - 建立工业帝国
   - 控制关键原材料供应
   - 影响市场价格

**优势：**
- ✅ 产品需求稳定
